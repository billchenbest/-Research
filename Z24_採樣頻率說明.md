# Z24資料集採樣頻率說明

## 重要澄清

### 採樣頻率 vs 奈奎斯特頻率

根據Z24資料集的實際資料：
- **時間間隔**：deltaT = 0.01 秒
- **採樣頻率**：fs = 1 / deltaT = 1 / 0.01 = **100 Hz** ✅
- **奈奎斯特頻率**：f_nyquist = fs / 2 = 100 / 2 = **50 Hz**

### 說明

1. **採樣頻率（Sampling Frequency）**：100 Hz
   - 這是資料的實際採樣頻率
   - 表示每秒採樣100個數據點
   - 計算方式：fs = 1 / deltaT = 1 / 0.01 = 100 Hz

2. **奈奎斯特頻率（Nyquist Frequency）**：50 Hz
   - 這是根據採樣定理確定的最大可解析頻率
   - 計算方式：f_nyquist = fs / 2 = 100 / 2 = 50 Hz
   - 根據奈奎斯特-香農採樣定理，只有頻率低於奈奎斯特頻率的信號才能被正確重建

### 驗證

從實際資料檔案驗證：
```
檔案：01D0003.aaa
行1: 01B2303.aaa
行2: 65536          (樣本數量)
行3: 0.01           (時間間隔，秒)
```

計算：
- 採樣頻率 = 1 / 0.01 = 100 Hz ✅
- 奈奎斯特頻率 = 100 / 2 = 50 Hz ✅

### 結論

**文件中寫的100 Hz是正確的採樣頻率。**

50 Hz是奈奎斯特頻率（最大可解析頻率），不是採樣頻率。兩者關係：
- 採樣頻率 = 100 Hz（資料實際採樣率）
- 奈奎斯特頻率 = 50 Hz（理論最大可解析頻率）
